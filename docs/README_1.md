## üõ†Ô∏è Technical Details

### Dependencies
- `qrcode`: Generates the sovereign gateway QR
- `ws`: WebSocket for real-time updates
- Pure Node.js HTTP server (no framework needed)

### Architecture
- QR encodes session token with Pioneer #001 designation
- WebSocket broadcasts state changes to all connected clients
- Dashboard updates in real-time without polling
- All data stored in memory (no database needed for demo)
