## Emergency Fixes

### Port 3000 in use?

```bash
./START_PRODUCTION.sh
```

### Missing dependencies?
```bash
npm install
./START_PRODUCTION.sh
```

### Can't see from phone?
1. Make sure on same WiFi
2. Use computer's IP, not localhost
3. Check firewall settings
